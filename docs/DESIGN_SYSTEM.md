# 设计系统

## 设计关键词

东方、克制、神秘、苹果感、仪式感、高级、流畅。

## 颜色

- 背景：深墨色、黑金、暗青。
- 点缀：金色、玉色。
- 文字：米白、浅灰。

## 动效

- 木棍分离动画：用于起卦流程，让用户感到每一步都在形成结果。
- 页面淡入：页面切换和主要模块进入时使用，保持安静、顺滑。
- 卡片浮动：用于重点信息卡片，幅度要小，避免干扰阅读。
- 按钮按压反馈：所有主要按钮都要有 hover、press、loading 或过渡状态。
- 卦象线条生成动画：展示成卦时，让阴阳线逐步出现，增强仪式感和理解感。

## 视觉原则

- 移动端优先，保证小屏阅读舒适。
- 卡片要克制，不要堆叠太多边框和阴影。
- 页面层级要清楚，先给结论，再给依据，再给原文和扩展内容。
- 东方感来自留白、节奏、文字和细节，不靠堆满符号。
- 高级感来自克制、准确和流畅，不靠强烈颜色和夸张动画。

## 禁止

- 大红大紫。
- 廉价玄学风。
- 过多闪烁。
- 过多弹窗。
- 花哨但没意义的动画。
- 用装饰压过内容。
